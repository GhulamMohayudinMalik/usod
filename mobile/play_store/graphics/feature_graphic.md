## Color Scheme

| Element | Color | Hex Code |
|---------|-------|----------|
| Background | Dark Gray | `#111827` |
| Primary Text | White | `#FFFFFF` |
| Accent 1 | Emerald | `#10B981` |
| Accent 2 | Cyan | `#06B6D4` |
| Highlight | Amber | `#F59E0B` |

## Typography

- **App Name**: Bold, 72-96pt
- **Tagline**: Regular, 36-48pt
- **Font Suggestions**: 
  - Inter
  - Roboto
  - SF Pro Display
  - Montserrat

## Key Design Elements

1. **Shield Icon** - Represents security
2. **Chart/Graph** - Represents analytics
3. **AI/Brain** - Represents AI insights
4. **Lock** - Represents secure access
5. **Network Node** - Represents monitoring

## Tips

âœ… **Do:**
- Keep text large and readable
- Use high contrast colors
- Keep important content in center
- Match app's dark theme
- Include app name prominently

- Use small text (under 16pt)
- Crowd the image
- Use screenshots (save for screenshot slots)
- Include device frames
- Put important content at edges

## Export Settings

- **Format**: JPEG or PNG (no transparency)
- **Quality**: 100% (maximum)
- **Color Profile**: sRGB
- **Resolution**: 72 DPI minimum
